# Frontend Frostbitten Tracker

Interface Next.js pour suivre les rares Frostbitten en local.

## Prerequis

- Node.js 20+
- Backend local lance sur http://localhost:4000

## Configuration

1. Copier .env.example vers .env.local
2. Verifier NEXT_PUBLIC_API_URL=http://localhost:4000

## Demarrage

```bash
npm install
npm run dev
```

Application disponible sur http://localhost:3000

## Build local

```bash
npm run build
npm run start
```
